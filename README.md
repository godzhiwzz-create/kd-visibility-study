# KD Visibility Study

This repository contains the public validation workspace for the KD
visibility-degradation study.

## Contents

- `paper_spic/`: manuscript availability note. The submission source and PDF
  are not tracked here during journal review.
- `faster_rcnn_kd_true/`: Faster R-CNN cross-architecture validation scripts.
- `m13_dense_check/`: dense mechanism-check scripts used for the M1/M3 trend
  reinforcement analysis.
- `requirements.txt`: Python package requirements for the validation scripts.

## Notes

Legacy code, duplicate paper folders, reference-paper PDFs, draft notes, and
temporary build outputs are intentionally omitted from the GitHub workspace.
The manuscript package is kept outside the public repository while the paper is
under review. The validation scripts assume the remote dataset and output paths
documented in their local README files.
