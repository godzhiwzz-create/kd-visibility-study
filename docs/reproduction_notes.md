# Reproduction Notes

The scripts in this repository are release-safe versions of the remote
experiment drivers. They assume the original remote workspace layout and do not
bundle data or weights.

## External Assets

Required but not tracked:

- Cityscapes / Foggy Cityscapes in YOLO format
- clear-weather YOLO teacher checkpoint
- Faster R-CNN teacher checkpoint generated by the true-KD queue
- remote output directories referenced by the queue and analysis scripts

The paper states that Foggy Cityscapes is available through the Cityscapes
dataset site. The dataset itself must be obtained under its own license.

## Script Groups

| Script directory | Purpose |
|------------------|---------|
| `scripts/yolo_dense_mechanism_checks/` | dense YOLO degradation checks for M1/M3 diagnostics |
| `scripts/faster_rcnn_boundary_check/` | Faster R-CNN true KD and M2 boundary-check runs |

## Expected Workflow

1. Prepare the remote dataset and teacher checkpoints.
2. Run the Faster R-CNN true-KD queue to create the teacher and representative
   branch results.
3. Run the M2 Faster R-CNN queue and summary script.
4. Run the dense YOLO mechanism checks and analysis script.
5. Compare generated summaries against `experiments/`.

The committed `experiments/` files are the paper-facing summaries used to audit
the submitted results.
