# Mechanism Dense Checks

This directory stores the data sources used for mechanism discrimination in the
IEEE Access submission.

Files:
- `m13_dense_metrics.csv`: five-level dense check for M1 and M3 using real runs.
- `m13_dense_summary.json`: Pearson correlation summary for M1 and M3.
- `m2_occlusion_loc.json`: existing five-point occlusion evidence used for M2.
