% Key observations paragraph
\begin{itemize}
    \item \textbf{遮挡悖论}: 无遮挡时KD方法优势显著 (+4.9\% for localization), 高遮挡时三种方法收敛至相同水平 ($\sim$0.479)
    \item \textbf{KD收益递减}: 随着遮挡率增加，KD的相对增益从4.9\%降至0\%
    \item \textbf{最优方法}: localization\_only在低遮挡表现最佳，logit\_only在中等遮挡略有优势
\end{itemize}
